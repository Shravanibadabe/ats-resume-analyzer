# ATS Resume Analyzer Pro

AI-powered ATS Resume Screening Tool built using Next.js, Tailwind CSS, Gemini AI, PDF/DOCX Parsing, and jsPDF.

## Features

- ATS Score Analysis
- PDF Resume Upload
- DOCX Resume Upload
- Keyword Matching
- Missing Keyword Detection
- Resume Statistics
- AI Resume Review
- Interview Readiness Score
- PDF Report Generation

## Tech Stack

- Next.js 15
- Tailwind CSS
- Gemini AI
- pdfjs-dist
- mammoth
- jsPDF
